Analyse_ngrams
==============

.. _n-grams:

Introduction
------------
The analyse_ngrams module will split an article into ngrams. It use `ngrams` from `nltk` library.

Usage
-----
To use the module::

>>>import coast_core
>>>analyse_ngrams.function(to_use)

or::

>>>import coast_core
>>>from coast_core import analyse_ngrams
>>>function(to_use)
