Introduction

Usage
-----

To use the module you can import 3 different tools *file*, *lang* and *measures*. Replace the *x* in the next commands by one of these keyword::

>>>import coast_core
>>>x_utils.function(to_use)

or::

>>>import coast_core
>>>from coast_core import x_utils
>>>function(to_use)
